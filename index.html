Ontario EV Ownership Analysis

I analysed a data set of registered EVs (Electric Vehicles) in Ontario, Canada. The data is broken down by FSA (A grouping of Postal Code regions). One potential use case of this dashboard is by EV repair business owners who may like see information as to where might be a good location to open up a EV repair shop in Ontario. The dashboard displays the areas with the most (and the least) EVs owned, as well as areas where there has been a greater increase (or deccrease) in EV ownership.
